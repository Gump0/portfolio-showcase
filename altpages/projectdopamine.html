Project Dopamine is an immersive sandbox game set inside a nostalgic computer interface, where curiosity leads you deep into a world of mini-games, hidden corners, and interactive experiences. But there's more beneath the surface...


This project was developed as a group effort within the span of roughly six months. As the groups technical lead I was in charge of building the game using the Unity engine, along with Unity's C# scriping framework.

I was also in charge with designing many of the games core technical systems, and user interfaces.
