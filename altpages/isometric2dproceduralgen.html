As a result of a long-term project I am currently developing, I have created this! A completly procedural tile-based world generation system. Designed to work in a 2D isometric space, this system includes many features such as infinite amounts of biomes, foliage, randomly spawned structures, NPC spawning, a convincing 3D-like collision system, and much more!


üå± Seed-Based Generation

Taking inspiration from other games that feature procedural generation. I've developed this algorithm with being seed-based in mind. Meaning that if the correct number value is stored, then the same world can be generated every time. With this in mind, we can create a living, breathing world. With a vast variety of diverse landscapes, biomes, and generated structures. It's as simple as changing a few values! To create convincing natural landscapes, this algorithm utilizes layering perlin noise to accomplish breathtaking landscapes that are different yet consistent every time.
